Mock the process method of ICM consumer so it does nothing.

This mocks the process method and removes all processing, but still catches the raised exceptions. If duration_s is provided, the process method is made to sleep for the given amount of seconds before returning, unless something raises an exception within that timeframe.

def mock_consumer_process_no_action_catch_exception(duration_s=0):
# 
    try:
        time.sleep(duration_s)
    except Exception:
        raise
